1. Description of Service

The Bot provides the following features via Discord slash commands and prefix commands:

  • QRIS payment generation (!qris) — generates dynamic QRIS payment QR codes for Indonesian e-wallet/bank transfers, configured per server via /qrissetup.
  • Robux/IDR conversion calculator (!calc).
  • Roblox group membership eligibility check (!check) — verifies whether a Roblox account has been a member of a designated Roblox group for the required duration.
  • Giveaway eligibility verification (!giveaway) — checks Discord server membership and Roblox group membership against giveaway requirements.
  • Order placement (!order) — submits Robux orders to our backend order-processing system on your behalf.
  • Payment proof submission (!payment) — uploads payment confirmation images to our backend for order verification.
  • Leaderboard display (!leaderboard) — displays top buyers based on cumulative Robux purchases.

The Bot is a tool that facilitates these functions; it does not itself hold custody of funds beyond generating payment instructions and relaying data to our backend and to Roblox's public APIs.
